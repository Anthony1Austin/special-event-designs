# Special Event Designs – Logo assets

Crisp, legible logo recreated as vector SVGs for use across the website and other environments. All files are in this folder.

## Design types (6)

| # | Name | Description | Use |
|---|------|-------------|-----|
| 01 | **Full lockup** | Emblem + “SPECIAL EVENT DESIGNS” + tagline (horizontal) | Hero, footers, print |
| 02 | **Name only** | Emblem + “SPECIAL EVENT DESIGNS” (no tagline) | Header, general UI |
| 03 | **Emblem only** | Diamond “S” mark only | Favicon, app icon, small spaces |
| 04 | **Wordmark only** | “SPECIAL EVENT DESIGNS” text only | Minimal lockups, text-only contexts |
| 05 | **Compact** | Emblem + “SED” | Tight spaces, social, badges |
| 06 | **Stacked** | Full logo arranged vertically | Square formats, mobile |

## Background variants (for all types)

- **`-transparent`** – For light or colored backgrounds. Logo in brand colors (gold, dark, white).
- **`-on-black`** – For dark environments. Same logo with a black background rectangle; use on dark pages or in dark themes.

## File naming

```
sed-logo-[01–06]-[design-name]-[transparent|on-black].svg
```

Examples:
- `sed-logo-01-full-lockup-transparent.svg`
- `sed-logo-01-full-lockup-on-black.svg`
- `sed-logo-02-name-only-transparent.svg`
- … through 06, each with `-transparent` and `-on-black`.

## Colors (transparent versions)

- Gold: `#fbbf24` (emblem, tagline)
- Dark: `#1f2937` (inner diamond, wordmark on light)
- White: `#ffffff` (S in emblem)

On-black versions use white/gold for type and the same emblem; background is `#000000`.

## Website usage

- **Header:** Uses `sed-logo-02-name-only-transparent.svg` (or compact/05) for a clear horizontal lockup on the light header.
- **Footer / dark sections:** Use the `-on-black` variant of the same type when the background is dark.

All logos are SVG and scale without quality loss.
